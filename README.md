# Linux Kernel v0.01 클론 코딩 및 분석 프로젝트

이 저장소는 리눅스 커널의 첫번째 공개 버전, 
v0.01의 소스 코드를 직접 클론 코딩하며 
학습하는 과정을 기록하는 공간입니다.

## 🚀 프로젝트 소개

13,000여 라인에 달하는 코드를 직접 입력하며, 
코드에서 쓰인 메서드 및 함수, 로직 등에 대해 
주석을 달아가며 코드를 깊이있게 탐구하는 것을 
목표로 합니다.
